### How do you loop through or enumerate a JavaScript object?

You can loop through an object using `for...in` or `Object.keys()` combined with `forEach()`.

```javascript
const obj = {name: 'John', age: 30};
for (let key in obj) {
  console.log(key, obj[key]);
}
```
