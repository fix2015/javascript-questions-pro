### How do you empty an array?

There are multiple ways to empty an array in JavaScript. You can set its length to `0`, use `splice()`, or create a new empty array.

Example:

```javascript
let arr = [1, 2, 3];
arr.length = 0;
console.log(arr);  // Output: []
```
