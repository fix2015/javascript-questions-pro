### What is an Unary operator?

An unary operator is an operator that operates on a single operand. It performs operations like negation, increment, and decrement.

Examples of unary operators include:

- `+` (Unary plus)
- `-` (Unary minus)
- `++` (Increment)
- `--` (Decrement)
- `!` (Logical NOT)

Example:

```javascript
let a = 5;
console.log(++a); // 6
console.log(a++); // 6
console.log(a); // 7
```
