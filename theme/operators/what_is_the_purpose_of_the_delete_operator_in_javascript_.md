### What Is the Purpose of the `delete` Operator in JavaScript?

The `delete` operator is used to remove a property from an object or an element from an array.

```javascript
let obj = { name: 'John', age: 30 };
delete obj.age;
console.log(obj); // { name: 'John' }
```
