### What are the bitwise operators available in JavaScript?

JavaScript supports several bitwise operators, including AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), and shift operators (`<<`, `>>`, `>>>`).

Example:

```javascript
const a = 5; // 0101
const b = 3; // 0011
console.log(a & b); // 1 (0101 & 0011 = 0001)
```
