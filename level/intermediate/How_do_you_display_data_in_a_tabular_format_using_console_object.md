### How do you display data in a tabular format using console object?

You can use the `console.table()` method to display data in a tabular format. This is especially useful when dealing with arrays of objects.

Example:

```javascript
let arr = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 }
];
console.table(arr);
```
