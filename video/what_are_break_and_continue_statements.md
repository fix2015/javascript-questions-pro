### What are break and continue statements?

- `break` is used to exit a loop prematurely.
- `continue` is used to skip the current iteration and proceed to the next iteration of the loop.

Example:

```javascript
for (let i = 0; i < 5; i++) {
  if (i === 3) break;
  console.log(i); // 0, 1, 2
}
```
