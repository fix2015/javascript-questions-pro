### What would be the result of 1+2+'3'?

In JavaScript, `1 + 2 + '3'` results in `'33'` because the `+` operator first adds the numbers (`1 + 2 = 3`) and then concatenates the result with the string `'3'`.

Example:

```javascript
console.log(1 + 2 + '3'); // '33'
```
